# Installation

## Adding Japit to your project

To install Japit, you can use the package manager of your choice:

<Tabs>
<TabItem value="npm" default>
```bash
npm install japit
```
</TabItem>
<TabItem value="yarn">
```bash
yarn add japit
```
</TabItem>
<TabItem value="pnpm">
```bash
pnpm add japit
```
</TabItem>
</Tabs>
